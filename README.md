##Базовые команды
### Клонирование
```sh
git clone https://github.com/EvgenyFedorov/testVeneziaStone.git .
```

### Установка зависимостей
```sh
composer update
```

### Настройка .env
```sh
переименовать .env.example в .env и добавить свои реквизиты MySQL
```

### Миграции
```sh
php artisan migrate
```

### Интерфейс
```sh
Открывается по корневому пути, например test.loc/
```
